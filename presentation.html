  • Component: Camera.tsx
  • Technology: MediaStream API
  • Resolution: 1280x720 (ideal)
  • Features: Multi-device support, multi-frame capture for liveness

Stage 2: Vision Analysis

  • Service: geminiService.ts
  • Model: Gemini 2.5 Flash (multimodal)
  • Innovation: Semantic embeddings instead of traditional vectors
  • Output: JSON with bounding box, description, quality score
